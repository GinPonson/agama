# agama
agamga是一个轻量易用的Java爬虫框架。

# 特点
- [x] 使用流式Api，方便易用
- [x] 使用自定义的xpath注解+POJO方式爬取
- [x] 支持多线程
- [x] 支持Redis实现分布式抓取
- [x] 支持抓取Ajax加载的页面
- [x] 支持解析Json形式的Api数据
